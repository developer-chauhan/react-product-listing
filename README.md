# react-redux

### Tech

Used a number of packages for scalable applications:

```sh
$ git clone
$ npm install
$ npm start
```
if you are using yarn
```sh
$ git clone
$ yarn install
$ yarn start
```

Open your favorite Terminal and run these commands.

#### serve
```sh
$ npm start
```

```sh
$ yarn start
```
